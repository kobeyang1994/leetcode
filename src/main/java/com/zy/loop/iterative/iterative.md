### Iterative Method

**什么是迭代法？**

不断的通过久的变量值，递推计算新的变零值。

**迭代法的应用：**

- 求数组的精确或者近似解
  - 二分法
  - 牛顿迭代法
- 在一定范围内查找目标
  - 二分查找
- 机器学习中的迭代
